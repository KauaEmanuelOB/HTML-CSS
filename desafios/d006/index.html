Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Linha horizontal
  • Quebra de linha no parágrafo
  • Negrito
  • Itálico ou ênfase
  • Marcaçao de texto
  • Texto sobrescrito
  • Texto subscrito
  • Preservar os espaçamentos dentro do código.
